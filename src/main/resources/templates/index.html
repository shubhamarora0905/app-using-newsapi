Developed by Shubham Arora

This is a simple application made using Spring Boot, Postgres Database, JPA & hibernate. The UI is made using thymeleaf and Semantic UI.

It uses heroku connect to sync data between Postgres DB and Salesforce.

So, whenever you save or edit a contact, the same data is saved to Salesforce.
